fal.ai API key

Alcon AI Studio runs on fal.ai — one key unlocks every model (Seedance, Kling, Vidu, Hailuo, FLUX, Seedream, Imagen). Create a key at fal.ai/dashboard/keys, paste it below and you're live.

This is the browser edition. Your key is stored only in this browser (localStorage) and sent directly to fal.ai from your device — it is never uploaded to GitHub or any Alcon server. Use your own key on your own machine. For a shared/production deployment where the key stays hidden server-side, run the included Node server instead (see the repo's DEPLOY.md).
