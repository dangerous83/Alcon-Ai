fal.ai API key

Alcon AI Studio runs on fal.ai — one key unlocks every model in the platform (Seedance, Kling, Vidu, Hailuo, FLUX, Seedream, Imagen). Create a key at fal.ai/dashboard/keys, paste it below and you're live. The key is stored server-side only and never exposed to the browser.

Platform

  • Video — up to 15 seconds per shot, start-frame & end-frame keyframing, 1080p output.
  • Image — up to 4 outputs per run, multi-reference editing, up to 4K-class detail.
  • Storage — all finished media is mirrored to this server (data/media/), so results never expire.
  • Queue — generations run on fal.ai's GPU queue; the server tracks each job and resumes tracking after restarts.
